# Distro Grub Themes

A pack of GRUB2 themes for each Linux distribution
 
## Supported distributions
- Arch Linux
- Debian
- ElementaryOS
- Fedora
- Linux Mint
- Manjaro
- openSUSE
- Pop! OS
- Solus
- Ubuntu
